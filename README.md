## GitHub Pages 배포
1) GitHub에 새 repo 생성 (Public 추천)
2) 이 폴더(github_pages) 내용을 repo 루트에 업로드
3) Settings → Pages
4) Source: Deploy from a branch
5) Branch: main / (root)
6) Save

접속 URL:
- https://<username>.github.io/<repo-name>/
